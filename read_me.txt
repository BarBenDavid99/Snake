The snake game presents the old and well-known game.
The score is determined by the length of the snake.
In this game there are improvements to the game experience by using sound effects, record keeping by local storage so that the user has a goal to break his personal record or the record of his friends.
 The game is addictive and suitable for all ages.

 u can move the snake with arrows keys and freeze the game by pressing ESC
 built with html, css and js.
